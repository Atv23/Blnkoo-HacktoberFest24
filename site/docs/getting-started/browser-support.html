Browser Support

Blnkoo uses Autoprefixer to make most styles compatible with earlier browsers and Normalize.css for CSS resets. Blnkoo is designed for modern browsers. For best compatibility, these browsers are recommended:

  • Chrome Last 4
  • Microsoft Edge Last 4
  • Firefox Extended Support Release
  • Safari Last 4
  • Opera Last 4
  • Internet Explorer 10+

Blnkoo supports Internet Explorer 10+, but some HTML5 and CSS3 features are not perfectly supported by Internet Explorer.
